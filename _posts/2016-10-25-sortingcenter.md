Project delivery day with Herna Verhagen (PostNL CEO)

On this project, I was on charge of the whole software. 

**Features implemented**:

* Motor control
* Collision avoidance system
* Path planning
* Robot navigation
* Parcel detection
* Real time user feedback
* Basic GUI
* Multi-robot environment
